#  Echo – Chat UI Interface

Echo is a clean and responsive chat interface built using Next.js, TypeScript, and Tailwind CSS. It simulates a conversational AI experience using a mock backend API.

---

##  Features

-  Keyword-based intelligent responses (Next.js, React, MongoDB, etc.)
-  Real-time chat interface with user and bot messages
-  Send messages via button or Enter key
-  Typing indicator for bot responses
-  Clean and responsive UI (desktop + mobile)
-  Auto-scroll to latest message
-  Mock API with simulated delay (1–2 seconds)

---

##  Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **Backend:** Next.js API Routes (Mock API)
